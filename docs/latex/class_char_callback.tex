\doxysubsection{Descrição detalhada}
Callback para escrita em características BLE. Atualiza variáveis globais. 


\begin{DoxyItemize}
\item Interpreta strings recebidas do App via BLE.
\item Atualiza variáveis globais (nome e senha do Wi-\/\+Fi, telefone, setpoint, intervalo).
\item Chama a função \doxylink{funcoes_8cpp_a18165d9dc3d16f2b88e24905b7706472}{Salva\+Config()} para persistência imediata na NVS. 
\end{DoxyItemize}
